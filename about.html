About this project

The games on this site are constructed by html table elements and modifying the colors or text content within with Javascript and CSS.

  • LightSwitch is a game inspired by Lights Out. It's a common game for beginner programmers to code.
  • Tic-Tac-Toe is another simple game to program. I used this game to practice with handling form data.
  • Mastermind is a classic game that is more complicated than the others. I used the layout and colors of Wordle, which has numerous spin-off games made by fans such as Quordle.

Visit this project's GitHub Repository to see the code used to create this site.
